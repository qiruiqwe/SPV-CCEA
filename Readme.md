# Electric Vehicle Charging Infrastructure Planning

This repository contains the source code and datasets for the manuscript titled:
**"Survival Probability Vector-Guided Cooperative Co-evolutionary Algorithm for Large-Scale Electric Vehicle Charging Infrastructure Planning"**.

## Overview

This project provides a simulation and optimization framework for large-scale, multi-scenario charging infrastructure planning (CIP). The core problem involves jointly optimizing the discrete siting and integer sizing of electric vehicle charging stations across thousands of candidate locations, dealing with complex spatiotemporal traffic flows and queuing dynamics.

## Requirements

To run the simulation and experiments, the following environments are generally required:
- Python 3.8+
